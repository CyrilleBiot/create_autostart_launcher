# create_autostart_launcher

Juste un utilitaire pour créer un fichier de démarrage automatique d'application dans un environnement X.


## Dépendances

 python3, python3-gi, python3-pil

## Fichier crée

Le fichier sera créé dans 

```.config/autostart/```
